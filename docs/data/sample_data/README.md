# Sample data

Folder przeznaczony na przykładowe dane testowe:
- zanonimizowane faktury,
- przykładowe pliki PDF,
- obrazy testowe do OCR.

## Zasady
- nie przechowujemy tutaj prawdziwych danych wrażliwych,
- używamy tylko danych testowych lub zanonimizowanych,
- pliki mają służyć do demonstracji i testów działania systemu.
